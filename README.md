# Movie_Insights
Repositorio creado como parte del proyecto 2 del curso BD-143 - Programación II. Donde se busca desarrollar un sistema, en Python y con Programación Orientada a Objetos, que  ingeste datos de películas desde archivos CSV, que permita consultas, análisis  exploratorio (EDA) y visualización de gráficos o de manera interactiva mediante un  dashboard.
